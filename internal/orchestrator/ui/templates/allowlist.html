How allowlists are applied

  • Global entries apply to every package's runs.
  • Package entries apply only to runs of that package. Useful when one noisy dependency has a known internal destination.
  • Three kinds match three Differ categories:
    • cidr → suppress net_new_destination when the IP is in the range
    • path → suppress fs_new_path_read/write when the normalized path has the prefix
    • sni → suppress net_new_https_host when the SNI matches (case-insensitive)
  • The hardcoded DefaultCDNAllowlist (Cloudflare/GitHub/Google/Fastly/CloudFront) still applies underneath — adding entries is additive, never subtractive.
